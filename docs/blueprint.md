# **App Name**: AIMA Digital Innovations

## Core Features:

- Parallax Scrolling Landpage: Implements a smooth parallax scrolling effect to create an engaging user experience, drawing inspiration from Minale & Mann and MOAA.co.nz.
- Hero Section Animation: Replicates the initial animation sequence of Minale & Mann, where the company name 'AIMA' appears first, followed by a captivating image transition to the cover image of 'The Edge' building in Amsterdam.
- Services Showcase: Presents AIMA's core services (BIM, Consulting, Training, and Software Development) in a visually appealing manner, similar to the layout and style found on EngWorks Global's website.
- Content Translation: Translates all content of the landing page to the English language for a broader audience.
- Quality Vector Graphics: Incorporates high-quality vector images in the lower sections of the landpage to enhance visual appeal, taking cues from the style used on Bonsai BIM's website.

## Style Guidelines:

- Primary color: Deep Blue (#1A237E) to represent professionalism, trust, and technological expertise within the BIM and software development context.
- Background color: Very light blue-gray (#F0F4F8), providing a clean and modern backdrop that highlights the content and avoids distracting from the imagery.
- Accent color: Soft Lavender (#C5CAE9), used for interactive elements and highlights, complementing the primary blue with a touch of creativity and innovation.
- Font: 'Inter', a sans-serif typeface that combines a clean modern appearance with excellent readability, ideal for both headings and body text across the landpage.
- Note: currently only Google Fonts are supported.
- Use clean, modern icons that are related to BIM, construction, and software development to represent the services offered.
- Ensure a clean and structured layout, providing intuitive navigation and content consumption across various sections of the webpage, facilitating parallax scrolling and engaging user experience.
- Employ subtle animations for a modern feel on buttons.
